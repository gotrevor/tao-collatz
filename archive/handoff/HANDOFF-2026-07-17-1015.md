## The proven per-node recipe (used 18 times, green first-pass almost every time)

   `X_<name> := <max-tree>` with obtained locals replaced by explicit upstream `X_*` names.
2. Convert the `_atC` into `_atCX`: `obtain ⟨x᷀, h⟩ := foo_atC` → `have h := foo_atCX;
   set x᷀ : ℝ := X_foo`, then `rw [show X_<name> = <tree-at-set-locals> from rfl]` replacing
   the `refine ⟨…⟩`, then `intro …`. Body stays verbatim.
3. Re-add the old `_atC` as the one-line delegate `⟨X_<name>, <name>_atCX⟩` (statement
   character-identical).
4. Gotchas hit repeatedly: (a) don't orphan a docstring — two `/--` in a row is a parse
   error; merge or move; (b) `set_option maxHeartbeats … in` / `open Classical in` attach to
   the NEXT declaration — insert new defs ABOVE those lines, never between them and their
   theorem; (c) `set` pinned numeric constants BEFORE `have`-ing hypotheses whose statements
   carry the same literals only where the original did so.
