    §7.4: the first-passage unrolling of the Q recursion (paper (7.45), node X8/X9 entry) #

    Paper anchor: Tao 2019 (7.44)–(7.45), p.47. The paper iterates (7.35) up to the stopping time k = first time the cumulative l-increment exceeds s, obtaining

    Q(j,l) = E [ exp(-ε³ Σ_{i<k} 1_W((j,l)+v_{[1,i]})) · Q((j,l)+v_{[1,k]}) ].     (7.45)

    D6 finitization: no infinite iid sequence / stopping-time measure theory. Instead we define the stopped value Qstop s j l by well-founded recursion on the remaining height budget s : ℕ — each hold step spends its (positive) second coordinate from s, and a step that overshoots (d₂ > s, the paper's first passage l_{[1,k]} > s) lands on the un-stopped Q. The identity Qstop_eq : Qstop s j l = Q j l (any s) is then (7.45) verbatim, proved by strong induction on s over Q_rec.

    Case 2 (X8, (7.46)–(7.51)) and Lemma 7.9 (X9) both consume this operator: the overshoot branch is exactly "the walk exits the triangle through the top edge", and its endpoint value is what Q_le_Qm / the white-exit bound (7.51) control.

    • hold_support_snd_ge / hold_zero_of_snd_lt — hold's second coordinate is ≥ 3 ((k, 3 + Σ pascalNe3 increments)), so every step spends height and the budget recursion terminates.
    • Qstop — the stopped value.
    • Qstop_eq — the (7.45) identity.
