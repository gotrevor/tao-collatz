    theorem TaoCollatz.tao_collatz (f : ℕ → ℝ) (hf : Filter.Tendsto f Filter.atTop Filter.atTop) :
    AlmostAllPos fun (N : ℕ) => ↑(colMin N) < f N

    Theorem 1.3 (Tao 2019): for any f : ℕ → ℝ with f(N) → ∞, almost all N (in logarithmic density) satisfy Colmin(N) < f(N).

    theorem TaoCollatz.tao_collatz_quantitative :
    ∃ (c : ℝ) (C : ℝ), 0 < c ∧ 0 < C ∧ ∀ (N₀ x : ℕ), 2 ≤ N₀ → 2 ≤ x → 1 - C / Real.log ↑N₀ ^ c ≤ logProb {N : ℕ | colMin N ≤ N₀} (Finset.Icc 1 x)

    Theorem 3.1 (Tao 2019, Colmin form): quantitative version — the log-probability that Colmin(N) ≤ N₀ on the window [1, x] is at least 1 - C/(log N₀)^c.
