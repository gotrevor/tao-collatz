    TRUSTED BASE — the main theorem statements #

    This file is the only trusted surface of the library (BLUEPRINT §3), a three-statement surface: Theorem 1.3 and Theorem 3.1 of Tao 2019 (arXiv:1909.03562) are the paper's, stated from first principles; tao_collatz_quantitative_explicit (with the constant cTao) is OUR augmentation beyond the paper — the paper proves ∃ c and Remark 1.4 gives only a shape, never a value. TaoCollatz imports here are ONLY Basic.Collatz + Basic.LogDensity (elementary defs: col, colMin, log density via Finset sums and Tendsto); the mathlib Pow.Real import supplies just the rpow notation used in Theorem 3.1's error term.

    theorem TaoCollatz.tao_collatz (f : ℕ → ℝ) (hf : Filter.Tendsto f Filter.atTop Filter.atTop) :
    AlmostAllPos fun (N : ℕ) => ↑(colMin N) < f N

    Theorem 1.3 (Tao 2019): for any f : ℕ → ℝ with f(N) → ∞, almost all N (in logarithmic density) satisfy Colmin(N) < f(N).

    theorem TaoCollatz.tao_collatz_quantitative :
    ∃ (c : ℝ) (C : ℝ), 0 < c ∧ 0 < C ∧ ∀ (N₀ x : ℕ), 2 ≤ N₀ → 2 ≤ x → 1 - C / Real.log ↑N₀ ^ c ≤ logProb {N : ℕ | colMin N ≤ N₀} (Finset.Icc 1 x)

    Theorem 3.1 (Tao 2019, Colmin form): quantitative version — the log-probability that Colmin(N) ≤ N₀ on the window [1, x] is at least 1 - C/(log N₀)^c.
