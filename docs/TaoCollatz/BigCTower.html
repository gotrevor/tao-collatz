    A readable tower ceiling for the assembled constant #

    This file deliberately favors a short public statement over a tight bound. The tenTower vocabulary and its calculus live in Basic/ExplicitConstants.lean (a Mathlib-only leaf below the trusted surface, so Statement.lean can state the headline); this file chases the development's leaf constants up to C_tao_assembled ≤ tenTower 62.
